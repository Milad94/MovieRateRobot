# Movie Rate Robot

## Step 1:
Create a database in MySQL or PostgresSQL

## Step 2:

Create a local_config.py like this:

```python
ATABASE_CONFIG = {
    'user': ' Your username',
    'passwd': 'Your password',
    'database_name': 'Your database name',
    'host': 'localhost',
    'port': 3306
}
DATABASE_TYPE = "[MYSQL or POSTGRESSQL]"

```


## Step :
Run main.py
